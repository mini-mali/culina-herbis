## Get started

### Download from GitHub
```
$ git clone https://github.com/mini-mali/culina-herbis.git
$ npm install
```


### Build SASS (run first time only)
```
$ npm run build
```


### Start local server [live reload HTML, SASS]  (http://127.0.0.1:8080/)
```
$ npm run start
```
